# Philosophy of Propagation Logic

**Process First — Substance Never**

Propagation Logic is not another formal system added to the existing toolkit of mathematics and logic. It is an explicit rejection of the substance ontology that has dominated Western thought since Aristotle and a return to **process as the fundamental reality**.

### Core Ontological Commitment

In PL, there are no static objects, no eternal propositions, no completed sets, and no timeless numbers that “exist” independently.  
The only primitives are **propagating patterns** engaged in continuous **becoming**.

A pattern `P = (v_P, L_P)` is not a thing that *has* a value and an extra piece of data called “load.”  
The **informational load** `L_P` **is** the accumulated history and intensity of the propagation process itself. It is the living trace of what the pattern has undergone. The designation component `v_P` is only a transient stabilization — a momentary “being” that emerges from the flux.

“Being” is therefore secondary and derivative. **Becoming** is primary.
